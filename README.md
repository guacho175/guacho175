## 🚀 Proyectos destacados

### 🗺️ [Microbasurales](https://github.com/guacho175/microbasurales)
Plataforma web municipal desarrollada en **Django** para registrar, visualizar y gestionar denuncias de microbasurales con geolocalización GPS, carga de imágenes y seguimiento de estados. Incluye autenticación de usuarios y panel administrativo.

`Python` `Django` `PostgreSQL` `HTML/CSS` `JavaScript` `Geolocalización`

---

### 💡 [Sistema Contaminación Lumínica](https://github.com/guacho175/sistema_contaminacionLuminica)
Sistema de monitoreo del Decreto Supremo N°1 del Ministerio del Medio Ambiente para el control de la contaminación lumínica en la Región de Coquimbo. Backend con Python y visualización de datos.

`Python` `SQL` `Análisis de datos` `Normativa ambiental`

---

### 📱 [TuBarrioLimpio App](https://github.com/guacho175/guacho175-TuBarrioLimpioApp)
Aplicación móvil para reporte ciudadano de microbasurales, complemento del sistema web. Desarrollada en Kotlin para Android.

`Kotlin` `Android` `Móvil`
